Use this evaluation tool to check how closely your technology project or programme is aligned with the Technology Code of Practice.

Who will use this tool

Service or project managers can use this tool to:

  • determine if you are developing your technology project or programme in accordance with the Technology Code of Practice
  • create a report to share with others
  • do an internal evaluation in advance of an assessment such as Spend Control

Assurance teams can use this tool to:

  • determine if the project or programme is developing in accordance with the Technology Code of Practice
  • create a report to provide recommendations and support

Before you start

You will need a good understanding of the technology project or programme. You can save your evaluation at any point and reload it. Your evaluation will not be sent anywhere until you choose to share it.
